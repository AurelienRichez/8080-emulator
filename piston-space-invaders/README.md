
This program emulates a space invader arcade game from 1978 with [piston](https://www.piston.rs/).


Commands
-------
 
 | key         | function                  |
 |-------------|---------------------------|
 | enter       | insert coin               |
 | ctrl left   | start game with 1 player  |
 | ctrl right  | start game with 2 player  |
 | space       | fire                      |
 | arrow right | go right                  |
 | arrow left  | go left                   |
